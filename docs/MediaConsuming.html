MediaConsuming

Everything that implements the MediaConsuming Interface has methods to add and remove media

Methods

addMedia(m, idopt)

Parameters:
Name Type Attributes Default Description
m MediaStream | MediaStreamTrack

A media Object to add. This should be a MediaStream or MediaStreamTrack
id string <optional>
 m.id

A unique identifier for the added media. If none is given, this defaults to the medias id, which works fine local, but not on remote media since media ids are not synced with the remote peer

removeMedia(mopt)

Parameters:
Name Type Attributes Description
m MediaStream | MediaStreamTrack | string <optional>

The media to remove. Can be the actual media Object that was given or its id. If omitted or '*', all media that has been added is removed
