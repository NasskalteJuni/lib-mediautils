Properties
Name Type Attributes Default Description
template string <optional>

Path to the page template that the browser shall open. Works best with absolute paths
scripts Array <optional>

A list of paths to scripts that shall be loaded into the open web page
globals Object <optional>

A dictionary of globals to load into the web page opened by the browser. The key will be the global constants name
ignoreScriptOrder boolean <optional>
 false

Can be used to reduce the load- and start-time, if the order of the scripts does not matter. Otherwise, the scripts given will be loaded in the order they were given
