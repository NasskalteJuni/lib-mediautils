ConnectionManager(config, signaler, iceServersopt, useUnifiedPlanopt, verboseopt, loggeropt)

new ConnectionManager(config, signaler, iceServersopt, useUnifiedPlanopt, verboseopt, loggeropt)

create a new peer connection manager who handles everything related to transmitting media via RTCPeerConnections

Parameters:
Name Type Attributes Default Description
config Object
Properties
Name Type Description
name string

The name or identifier of this peer
signaler Signaler

The Signaler to transmit messages to the server
iceServers Array <optional>
 []

An array of ice servers to use, in the common RTCIceServers-format
useUnifiedPlan boolean <optional>
 true

Use of standard sdp. Set to false, Plan-B semantics are used but are not guaranteed to work on the given browser, therefore this is discouraged
verbose boolean <optional>
 false

Any action or step in the connection process can be logged, if this flag is set to true
logger console <optional>
 console

A logger that must offer the methods .log or .error. Only used in verbose mode, defaults to console
